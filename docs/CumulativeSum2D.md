## Abstract

長方形領域に対する累積和を処理するライブラリ。
構築時に元サイズの外側に1マス拡大する。

## Function

- set : 単一要素に値を代入する。
- add : 長方形領域に値を加算する。
- build : 累積和配列を構築する。
- query : 長方形領域の累積和を取得する。
- max / min : 累積和配列全体の最大値 / 最小値を取得する。
