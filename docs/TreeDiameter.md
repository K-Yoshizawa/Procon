# Tree Diameter - 木の直径

$n$ 頂点の木 $T$ の直径に関するライブラリです。

## Function

### Constructor

```
TreeDiameter(Graph<CostType> &tree)
```

- 木 $T =$ `tree` の直径を求めます。

**制約**

- `tree` は木

**計算量**

- $\textrm{O}(n)$

---

### Diameter

```
CostType Diameter() const
```

- 木 $T$ の直径を返します。

**計算量**

- $\textrm{O}(1)$
